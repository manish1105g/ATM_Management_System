# 🏧 ATM Management System

## 📌 Project Overview

The ATM Management System is a Python-based project designed to simulate the basic functionalities of an Automated Teller Machine (ATM). The system allows users to perform banking operations such as account balance checking, cash withdrawal, cash deposit, PIN verification, and transaction management in a simple and interactive way.

This project demonstrates the use of Python programming concepts, including functions, loops, conditional statements, and file handling, to build a real-world banking application.

---

## 🎯 Objectives

* Simulate real ATM banking operations
* Provide secure user authentication using PIN verification
* Allow users to perform transactions easily
* Manage account balance efficiently
* Improve understanding of Python programming concepts

## 💡 Functionalities

### 🔐 Authentication

Users must enter the correct PIN to access ATM services.

### 💰 Balance Check

Displays the current account balance of the user.

### 💸 Cash Withdrawal

Allows users to withdraw money while checking sufficient balance availability.

### 💵 Cash Deposit

Enables users to deposit money into their account.

### 🚪 Exit System

Provides an option to safely exit the ATM system.
